## Develop the project

Run `npm install` once to install dependencies.  

- Start development-only server
  ```sh
  npm dev
  ```

- Start server
  ```sh
  npm start
  ```

- Lint project
  ```sh
  npm lint
  # To fix errors directly:
  npm lint:fix
  ```

- Format project
  ```sh
  npm format
  ```
